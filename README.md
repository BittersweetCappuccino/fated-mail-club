# Fated Mail Club

> A dark, elegant website for a fictional romantasy snail-mail subscription club — built with vanilla HTML/CSS/JS and a hand-drawn animated SVG envelope.

**View the interactive mockup →** https://bittersweetcappuccino.github.io/fated-mail-club/

Fated Mail Club is a concept website for a snail-mail subscription service that delivers original monthly **romantasy** (romance + fantasy) stories by post. Each month, subscribers receive a handcrafted envelope containing an illustrated novelette, a wax-sealed keepsake, a letter from the author, and other tangible extras — stories you can hold, with no screens or algorithms involved.

## Features

- **Animated hero** with a hand-built SVG envelope (open flap, letter peeking out, wax seal, postmark, gold flourishes, and sparkle accents)
- **Archive section** showcasing past story boxes, each with a custom CSS/SVG illustration, price, contents, and description
- **How It Works** four-step walkthrough of the subscription ritual
- **Reader testimonials**
- **Footer** with categorized navigation links
- Smooth-scroll navigation between sections
- Scroll-triggered reveal animations
- Fully mobile-responsive layout
- Dark, book-lover aesthetic throughout

## Tech stack

- Vanilla **HTML**, **CSS**, and **JavaScript** — no frameworks, no build step
- **SVG** for the envelope and package illustrations
- Google Fonts: Playfair Display (display), Cormorant Garamond (body), DM Mono (labels)
- `IntersectionObserver` for scroll reveals

## Getting started

No build tools required. Clone the repository and open the HTML file in any modern browser:

```bash
git clone <your-repo-url>
cd fated-mail-club
open fated-mail-club.html
```

Or serve it locally:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000/fated-mail-club.html
```

## Design notes

The palette pairs a midnight/indigo base with ember, rose, and gold accents for a moody, romantic, book-lover feel. Typography leans editorial — a high-contrast serif display face against a refined serif body and a monospace for small uppercase labels.

## Disclaimer

Fated Mail Club is a fictional brand created for a design/portfolio project. The company, products, pricing, and testimonials are not real.
